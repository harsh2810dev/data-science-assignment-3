📌 Project Overview

This project focuses on performing an in-depth Exploratory Data Analysis (EDA) on the Global Terrorism Dataset, which contains information about terrorist incidents across the world from 1970 to 2017.

The main objective of this project is to analyze terrorism-related patterns, identify trends, and extract meaningful insights using data visualization and statistical analysis techniques.

🎯 Objectives
Analyze global terrorism trends over the years
Identify the most affected countries and regions
Study attack types and weapon usage patterns
Examine casualties and incident severity
Generate meaningful insights using EDA techniques
Visualize complex terrorism data in an understandable manner
❗ Problem Statement

Terrorism datasets are extremely large and contain complex information spread across multiple variables. Raw data alone is difficult to interpret and analyze effectively.

This project aims to solve this problem by:

Cleaning and preprocessing the dataset
Handling missing and inconsistent values
Performing structured data analysis
Discovering hidden trends and patterns
Converting raw data into actionable insights
